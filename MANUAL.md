# 🎙️ MeetLog 使い方ガイド

会議を録音して、AIで議事録を作成できるツールです。

---

## 📥 インストール方法

1. ダウンロードしたZIPファイルを解凍
2. `MeetLog.exe` をダブルクリックで起動

※インストール作業は不要です

---

## 🎤 基本的な使い方

### Step 1: 録音する

1. **MeetLog.exe** をダブルクリックして起動
2. **マイク** と **システム音声** を選択
   - マイク: あなたの声を録音
   - システム音声: PC上の音声（Zoom/Teams等の相手の声）
3. **⏺️ 録音開始** ボタンをクリック
4. 会議終了後、**⏹️ 録音停止** をクリック
5. 自動でMP3ファイルが保存されます

### Step 2: 議事録を作成する（NotebookLM連携）

1. **📝 NotebookLMで議事録作成** ボタンをクリック
2. Googleアカウントでログイン
3. 「**新しいノートブック**」をクリック
4. 「**ソースを追加**」→「**ファイルをアップロード**」
5. 録音したMP3ファイルを選択してアップロード
6. AIが自動で以下を作成します：
   - ✅ 文字起こし
   - ✅ 要約
   - ✅ アクションアイテム

## 🔘 ボタンの説明

| ボタン | 機能 |
|--------|------|
| ⏺️ 録音開始 | 録音を開始します |
| ⏹️ 録音停止 | 録音を停止して保存します |
| ⏸️ 一時停止 | 録音を一時停止します |
| ▶️ 再開 | 一時停止から再開します |
| 📝 NotebookLMで議事録作成 | NotebookLMを開きます |
| ☁️ Google Drive | Google Driveを開きます |
| 📁 フォルダを開く | 録音ファイルの保存先を開きます |
| ⚙️ | 設定画面を開きます |
| 🔄 | 録音履歴を更新します |

---

## ☑️ オプション

| オプション | 説明 |
|-----------|------|
| **MP3変換** | ONにすると、録音をMP3形式で保存します（推奨） |
| **正規化** | ONにすると、音量を自動調整します（推奨） |

---

## 📂 録音ファイルの保存先

録音ファイルは以下の場所に保存されます：

```
MeetLog.exe と同じフォルダ
└── recordings
    └── 20241204_143052  ← 日時のフォルダ
        └── output.mp3   ← 録音ファイル
```

**📁 フォルダを開く** ボタンで簡単にアクセスできます。

## ⚙️ 設定

**⚙️ ボタン** をクリックすると設定画面が開きます。

### マイク遅延調整
マイク音声とシステム音声のタイミングがずれる場合に調整します。
- **負の値（-50等）**: マイク音声を遅らせる
- **正の値（+50等）**: マイク音声を早める

### 最大録音時間
1〜4時間の範囲で設定できます。

## ❓ よくある質問

### Q: 相手の声が録音されない
**A:** 「システム音声」の設定を確認してください。
- 「ステレオミキサー」または「WASAPI」を選択
- Windowsの「サウンド設定」でステレオミキサーを有効化

### Q: 音量が小さい/大きい
**A:** 「正規化」オプションをONにしてください。
自動で適切な音量に調整されます。

### Q: MP3に変換されない
**A:** FFmpegが必要です。
https://ffmpeg.org からダウンロードしてインストールしてください。

### Q: 録音が途中で止まる
**A:** 設定画面で「最大録音時間」を確認してください。
デフォルトは2時間です。
